# 🇺🇦 Генератор актів прийому-передачі | Google Apps Script

Автоматизація щомісячного створення актів прийому-передачі виконаних робіт для гіг-спеціалістів (Дія Сіті).

> Замість 1-2 годин ручної роботи — 5 хвилин і одна кнопка.

## Що робить

- ✅ Копіює шаблон для кожного спеціаліста
- ✅ Підставляє номер акта (автоінкремент)
- ✅ Вписує останній день місяця
- ✅ Вставляє опис послуг
- ✅ Заповнює ціну та суму
- ✅ Назва місяця українською
- ✅ Сума прописом українською (у дужках)
- ✅ ПДФО та ВЗ — формули вже в шаблоні

## 3 варіанти скрипта

| Варіант | Файл | Для кого | Drive API |
|---------|------|----------|-----------|
| **А** — Особистий диск | `Code_A_Personal.gs` | Один HR, файли на «Мій диск» | ❌ Не потрібен |
| **Б** — Спільний диск | `Code_B_SharedDrive.gs` | Команда, файли на Shared Drive | ✅ v2 |
| **В** — Гнучкий | `Code_C_Flexible.gs` | Різні шаблони у спеціалістів | ✅ v2 |

### Яку версію обрати?

- **Варіант А** — якщо ви єдиний HR і тримаєте всі файли на своєму Google Drive. Найпростіший — не потребує жодних додаткових налаштувань.
- **Варіант Б** — якщо файли на спільному диску компанії (Shared Drive). Підтримує як Google Sheets, так і xlsx шаблони.
- **Варіант В** — якщо у різних спеціалістів різна структура шаблонів (номер акта в різних клітинках, додаткові поля тощо). Клітинки задаються через аркуш «Маппінг» без редагування коду.

## Швидкий старт

### 1. Створіть Google Sheet
Новий документ на [Google Sheets](https://sheets.google.com)

### 2. Додайте скрипт
**Розширення → Apps Script** → вставте обраний варіант Code + файл `Setup.gs`

> Для варіантів Б та В: у Apps Script натисніть **Сервіси (+) → Drive API → v2 → Додати**

### 3. Запустіть Setup
В Apps Script оберіть функцію `setupPanelSheets` → ▶️ Виконати

### 4. Створіть папки
Оновіть Google Sheet (F5) → меню **Акти → Створити структуру папок**

### 5. Завантажте шаблони
Покладіть xlsx-файли у папку «Шаблони» на Google Drive

### 6. Заповніть дані
На аркуші «Дані» вкажіть ПІБ, РНОКПП, ID шаблону, останній номер акта

### 7. Генеруйте!
Меню **Акти → Згенерувати акти за місяць** 🎉

## Структура аркушів

### Аркуш «Налаштування»
| Рядок | Параметр | Клітинка |
|-------|----------|----------|
| 2 | Місяць (1-12) | B2 |
| 3 | Рік | B3 |
| 4 | ID папки шаблонів | B4 |
| 5 | ID папки вихідних файлів | B5 |

### Аркуш «Дані»
| Колонка | Зміст | Оновлення |
|---------|-------|-----------|
| A | ПІБ спеціаліста | Один раз |
| B | Коротке ім'я (для назви файлу) | Один раз |
| C | РНОКПП | Один раз |
| D | ID шаблону (Google Sheet) | Один раз |
| E | Останній № акта | Автоматично |
| F | Опис послуг (з Gemini) | Щомісяця |
| G | Ціна (грн) | Якщо змінилась |
| H | Сума (грн) | Якщо змінилась |

### Аркуш «Маппінг» (тільки Варіант В)
| Колонка | Зміст |
|---------|-------|
| A | ID шаблону або `default` |
| B | Клітинка (наприклад D2, K5) |
| C | Тип: `act_number` / `date` / `services` / `price` / `total` / `month` / `words` / `custom` |
| D | Для `custom`: текст або `col:Назва колонки` |

## Де знайти ID

**ID шаблону** — відкрийте Google Sheet шаблону, скопіюйте з URL:
```
https://docs.google.com/spreadsheets/d/ЦЕЙ_ЧАСТИНА/edit
```

**ID папки** — відкрийте папку на Google Drive, скопіюйте з URL:
```
https://drive.google.com/drive/folders/ЦЕЙ_ЧАСТИНА
```

## FAQ

**Меню «Акти» не з'явилося?**
Оновіть сторінку Google Sheet (F5).

**Google каже «Скрипт не перевірено»?**
Натисніть «Додатково» → «Перейти до проєкту» → «Дозволити».

**Помилка «не має доступу до документа»?**
Шаблон у форматі xlsx — використовуйте Варіант Б або В (вони конвертують автоматично). Або вручну відкрийте файл через Google Таблиці і використовуйте новий ID.

**Номер акта збився?**
Відредагуйте число в колонці E вручну.

**Як додати спеціаліста?**
Новий рядок на аркуші «Дані» + шаблон у папку «Шаблони».
